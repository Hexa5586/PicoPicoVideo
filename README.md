## 描述

一个搭建在本地服务器的视频播放与分享网站，支持视频的播放，上传，选择与搜索。



## 使用的工具及框架

开发工具：Visual Studio 2022, Adobe Dreamweaver 2021, Microsoft Edge

框架：Flask for Python, JinJa 2 for Python
